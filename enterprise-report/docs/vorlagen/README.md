# Vorlagen & Visualisierungen

Bilder/Mockups, wie Struktur und Berichte aussehen:

| Datei | Zeigt |
|-------|-------|
| `struktur-5-ebenen.svg` | Die 5 Ebenen (GF → Fachbereich → Themenbereich → Details, Historisierung quer) |
| `berichtsbaum.svg` | Alle 22 Fachbereiche (7 operativ, 15 Controlling) unter der GF-Wurzel |
| `bericht-layout.svg` | Wie ein Berichtsknoten aussieht: Lagebewertung + KPI-Karten + Detailtabelle |
| `architektur.svg` | Schichtenarchitektur (Module → Design → Kern → Provider → Quelle) |
| `instrumente-bcg.svg` | Controlling-Instrument: BCG-Portfolio (Star/Cash Cow/Question Mark/Poor Dog) |
| `massnahmen-board.svg` | Maßnahmen: SMART-KI-Empfehlung + Verwaltung/Tracking mit Alerts |
| `self-service-bi.svg` | Self-Service BI: Anforderung → KI-Beirat → Befunde + Maßnahmen |
| `bericht-vorlage.html` | Druckbare HTML-Vorlage eines Berichts (echte Design-Tokens; im Browser öffnen/drucken) |

SVGs in jedem Browser/Viewer ansehbar. Die HTML-Vorlage zeigt das reale Aussehen
(gleiche Farben/Ampel/Typo wie die App) und lässt sich als PDF drucken.
